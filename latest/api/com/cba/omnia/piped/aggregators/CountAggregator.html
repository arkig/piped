CountAggregator

case class CountAggregator[T]() extends Aggregator[T, Long, Long] with Product with Serializable

A simple count aggregator. It counts every value it sees. This aggregator exists so that it can be composed with other aggregators. If you are just after a count there are probably better ways than this.
